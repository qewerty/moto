Moto
====

Procedural 3D modelling and animation tools.

Requrements
-----------

 * GTK+-2.12
 * Cairo 1.6
 * GtkGLExt 1.2
 * Python 2.5 or 2.6

Build
-----

To build run SCons:

    $ scons

Or for debug version:

    $ scons config=config/debug.py
    $ export LD_LIBRARY_PATH=./build/linux2[-dbg]/lib

And run with:

    $ ./build/linux2[-dbg]/moto-gui

Installing
----------

Not supported yet.
